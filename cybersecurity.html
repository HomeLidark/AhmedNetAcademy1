اختبار الاختراق

🎯 مراحل اختبار الاختراق

من جمع المعلومات إلى استغلال الثغرات

المراحل الخمس
1. Reconnaissance (جمع المعلومات)

جمع معلومات عن الهدف باستخدام أدوات مثل Whois و Google Dorks.

2. Scanning (المسح)

مسح الشبكة والمنافذ باستخدام Nmap و Masscan.

3. Exploitation (الاستغلال)

استغلال الثغرات باستخدام Metasploit أو SQLMap.

4. Post-Exploitation (ما بعد الاستغلال)

الحفاظ على الوصول، جمع البيانات، وتوسيع النفوذ.

5. Reporting (التقرير)

توثيق الثغرات وتقديم توصيات الإصلاح.

أوامر المسح (Nmap)
# مسح سريع لمنافذ مفتوحة
root@kali:~# nmap -T4 -p- 192.168.1.1
Starting Nmap 7.94 ...
PORT STATE SERVICE
22/tcp open ssh
80/tcp open http
443/tcp open https
3306/tcp open mysql

# مسح متقدم مع اكتشاف الإصدارات
root@kali:~# nmap -sV -sC -O 192.168.1.1
Running: Linux 3.x
OS CPE: cpe:/o:linux:linux_kernel:3

# مسح الشبكة بالكامل
root@kali:~# nmap -sn 192.168.1.0/24
Nmap scan report for 192.168.1.1
Nmap scan report for 192.168.1.10

الهجوم والدفاع

⚔️ Attack vs Defense

افهم الهجمات لتتعلم كيف تحمي

أنواع الهجمات
Phishing
اجتماعي

خداع المستخدمين للحصول على معلومات حساسة عبر البريد الإلكتروني.

DDoS (Distributed DoS)
شبكي

إغراق الخادم بطلبات وهمية لتعطيل الخدمة.

Man-in-the-Middle
شبكي

اعتراض الاتصال بين طرفين وسرقة البيانات.

SQL Injection
تطبيقات

حقن أوامر SQL في نماذج الويب لاستخراج قاعدة البيانات.

Zero-Day
متقدم

استغلال ثغرة غير معروفة للشركة المطورة.

آليات الدفاع
Firewall (جدار الحماية)
شبكي

منع حركة البيانات غير المرغوب فيها بناءً على قواعد محددة.

IDS / IPS
شبكي

كشف ومنع الهجمات في الوقت الفعلي.

VPN (Virtual Private Network)
شبكي

تشفير الاتصال بين الأجهزة والشبكات.

SIEM
إدارة

جمع وتحليل السجلات لاكتشاف التهديدات.

Encryption (التشفير)
بيانات

تشفير البيانات أثناء التخزين والنقل (TLS/SSL, AES).

الأدلة الرقمية

🔍 Digital Forensics

جمع وتحليل الأدلة الرقمية

أدوات التحقيق
  • FTK Imager: إنشاء نسخة طبق الأصل (Image)
  • Autopsy: تحليل القرص الصلب
  • Volatility: تحليل الذاكرة (RAM)
  • Wireshark: تحليل الحزم الملتقطة
إجراءات التحقيق
  1. الحفاظ على الأدلة (Chain of Custody)
  2. إنشاء نسخة Image من القرص
  3. تحليل الملفات والـ Metadata
  4. استخراج الملفات المحذوفة
  5. تحليل سجلات النظام (Logs)
  6. كتابة تقرير للجهات القانونية
